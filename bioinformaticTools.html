Clique: an application for quantitative image color analysis
--
Clique is a simple desktop application I made to help with quantitative image analysis. Full disclosure: I am not a really a software developer, so this application might not be good compared to real software developer standards. Also you'll probably get some warnings when you try to download it (because I don't know how to make them go away). I really just made this as a quick and dirty application to help a student implement our image analysis pipeline.
