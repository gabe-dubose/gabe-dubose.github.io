Primary Questions:
--
Can random variant simulations generate proper null models of drift dynamics?
Are there better ways to measure selection in different contexts using population genomic data?
--
Background:
--
Now that generating population genomic data is becoming more accessible, there is an increasing need to find ways to incorporate this additional information into estimates of selection. I aim to develop and test computational approaches for generating null models that can be used to test hypotheses of selection.
--
A random variant simulation approach:
--
Currently, I am attempting to develop a variant simulation approach for generating null models of random drift dynamics. Here, thousands of random variants are simulated in an organism's reference genome, and a subsequent bootstrapping approach is used to randomly sub-sample these variants to generate a null model. I have applied this approach to two different studies aiming to measure selection in yeasts, and I am currently working on making this approach more generalizable.
